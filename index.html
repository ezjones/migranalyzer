How the analysis worked:

I have a python script running on Azure on a Linux VM. From there I grab the Weather Forecast and send it to Google SpreadSheets. There I create and publish these charts. The Headache Amount is the only data entered manually by me. I try to enter it manually several times a day. Eventually I will create the charts in python too and skip Gdocs but for now it works.
